## ⚠️ 常见问题解决

### 问题1：摄像头权限被拒绝
**解决**：
1. 首次点击扩展图标的"开始"按钮时，浏览器会弹出摄像头权限请求
2. 点击"允许"授予摄像头访问权限
3. 如果误点了拒绝，需要手动清除权限设置：
   - 访问 `chrome://settings/content/camera`
   - 找到相关网站权限并清除
   - 重新点击扩展的"开始"按钮

### 问题2：眨眼检测不准确
**解决**：
1. 确保光线充足
2. 面部正对摄像头
3. 调整与屏幕的距离（50-70cm）
4. 尝试更明显和完整的眨眼

### 问题3：扩展无法加载
**解决**：
1. 检查文件是否完整
2. 确认manifest.json格式正确
3. 重新下载依赖文件
4. 重启Chrome浏览器

### 问题4：视觉惩罚不生效
**解决**：
1. 检查扩展是否已启用
2. 确认内容脚本已加载
3. 查看控制台是否有错误
4. 尝试重新加载扩展
