## 🔬 技术实现

### 核心算法：EAR眨眼检测
```
EAR = (||p₂ - p₆|| + ||p₃ - p₅||) / (2 × ||p₁ - p₄||)
```

**检测标准：**
- 阈值：EAR < 0.25 判定为闭眼
- 防抖：连续2帧确认有效眨眼
- 频率：100ms检测一次，低功耗设计

### 架构设计
```
摄像头 → Offscreen文档 → MediaPipe分析 → Background转发 → Content Script渲染
```
